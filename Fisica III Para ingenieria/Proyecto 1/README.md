## Necesidad

En este desafío, se usará lo aprendido sobre electricidad y parte de magnetismo para crear un proyecto fotovoltaico, que permita generar no sólo prototipos de generación y almacenamiento eléctrico, también alternativas de integración de otros medios de generación eléctrica que sean integradas al hogar.
