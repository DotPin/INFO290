# Disciplina a desplegar

**Física III, Para Ingeniería**

***

## Descripción

Disciplina del área de la ingeniería, cual permite la comprensión de las leyes fundamentales de la mecánica clásica, a partir de una de sus ramas cual es la ciencia de las ondas electromagnéticas, estudiando leyes fundamentales como: Ley de Ohm, Kirchoft, Coulom, Superposición, Campo eléctrico, Gaus, Ampère, Faraday, Lenz, Maxwel, como las principales dentro de la disciplina, permitiendo comprender de manera teórica la naturaleza del fenómeno de la electricidad y magnetismo.


## Propuestas

- Caso de Uso 1: Desafío cual permite presentar un proyecto de generación eléctrica solar con sistema almacenamiento, cual ayuda como medio de respaldo eléctrico a servicios domiciliarios.
