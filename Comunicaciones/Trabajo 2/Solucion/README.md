## Descripción de Datos

Dentro de cada directorio, se almacena el directorio de códigos en Python como el informe que detalla los resultados del estudio de compresión de información para imágenes, generando un prototipo para procesos de transmisión de data en medios digitales.

- [Decodificador](https://github.com/DotPin/INFO290/tree/master/Comunicaciones/Trabajo%201/Solucion/Decodificador): Directorio cual almacena los archivos fuente en Python, para el proceso de compresión de imágenes.

- [Informe](https://github.com/DotPin/INFO290/blob/master/Comunicaciones/Trabajo%201/Solucion/informe%20proyecto1_%20comunicaciones.doc.pdf) Documento cual detalla los resultados del proceso de estudio en compresión de imágenes para su transmisión hacia medios digitales.
