# Disciplina a desplegar

**Fundamento de Ing de Software**

***

## Descripción

Disciplina del área de la ingeniería, cual introduce las nociones básicas de las teorías de la información, contemplando sus aspectos éticos y legales para su desempeño. Ello, beneficia en la aplicación de buenas práctias y estándares usando teorías, modelos y técnicas para el diseño de soluciones con enfoque de ingeniería orientada al software, permitiendo una base para la identificación y análisis de problemas para el transcurso de un ciclo de vida de softwares, cual fuerte es su documentación.

## Propuestas

- Caso de Uso 1: Desafío cual permite presentar un diseño procedimental de modelo unificación lógico (UML), para un proceso de venta.
