 
# Disciplina a desplegar

**Inteligencia Artificial**

***

## Descripción

Disciplina encargada de aprender mediante lenguaje de programación Python, algoritmos de aprendizaje automático de tipo supervisado y no supervisado, tales como Machine Learning u otros relacionados con Redes Neuronales, cuales fundamentan las bases de las áreas de la Inteligancia Artificial.

## Propuestas

- Caso de Uso 1: Desafío cual predende presentar un prototipo de simulación orientado en Aprendizaje Automático cual describe un problema del taxista. 

- Caso de Uso 2: Desafío cual presenta un algoritmo de Aprendizaje Automático que permite precedir si una persona tiene enfermedad crónica. 

- Caso de Uso 3: Desafío cual predende presentar un prototipo de software de simulación tipo agente biológico para ajuste de parámetros.

- Caso de Uso 4: Desafío cual presenta algoritmo de Aprendizaje Automático cual permite predecir si una persona es fumadora o no.

- Caso de Uso 5: Desafío cual predende presentar algoritmos de aprendizaje automático, que premiten generar estudio para el ajuste de parámetros astronómicos en la búsqueda de Galaxia, Estrellas o Quasares.
