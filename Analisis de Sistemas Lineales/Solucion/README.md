# Presentación Solución

Caso de estudio: Decodificador de Señal

***

## Necesidad

    Grupo de inteligencia cual se necesita para hacer el análisis de una señal proveniente de internet, donde los requisitos establecidos son decodificar el mensage a partir de señales producidas desde los 10Hz y ésta aumenta progresivamente con un delta=10Hz, produciendo en su resultado letras del alfabeto [A-Z] y números [0-9] respectivamente.
