# Presentación Caso de Transporte Valdiviano

Cliente, Transporte Público - Valdivia

***

## Necesidad

Se necesita un modelo de gestión de datos relacional, que permita el almacenamiento de la información orientada a los recorridos del transporte público de Valdivia.

## Implementacion

- SO Usado: Linux Kubuntu
- Gestor Base de datos: MySql
- [Quey's](https://github.com/DotPin/INFO290/blob/master/Base%20de%20Datos/Documentacion/querys) para implementación

## Consideraciones

- Gestor de bases de datos, puede usarse alguno disponible que soporte implementaciones relacionales
