## Descripción de Datos

- Archivos *.kml y *.kmz poseen datos de información geográfica de Valdivia, de las distintas lineas de transporte.
    - Información obtenida el 2019 por canales públicos. (Puede que ya no estén disponibles o cambien de ubicación).
    - Referencias del proyecto de transporte. [Enlace](https://www.dtpr.gob.cl/estudios/regionxiv).

- Mapa de Valdivia 1 sectorizado según los cuadrantes descritos ([2019](https://github.com/DotPin/INFO290/blob/master/Base%20de%20Datos/Datos%20a%20Compilar/Imagen2.jpg))

- Mapa de Valdivia 2 cual describe los recorridos de las líneas de buses en la comuna. [Enlace](https://github.com/DotPin/INFO290/blob/master/Base%20de%20Datos/Datos%20a%20Compilar/Mapa_Valdivia_enero_2014.pdf).

- Base de datos cual almacena la información requerida para la solución de la necesidad. [Enlace](https://github.com/DotPin/INFO290/blob/master/Base%20de%20Datos/Datos%20a%20Compilar/recorridos.sql).

## Consideraciones

Para la revisión del archivo de base de datos, es necesario usar algún software disponible en la red, que disponga acceso de éste archivo de tipo relacional.

### Recomendaciones

- SO Usado, Linux
- Gestor de Base de Datos, Sqlite
