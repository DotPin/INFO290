## Requerimientos de sistema

- SO Usado, Linux
- JFlex
- CUP
- JavaJDK

## Consideraciones

Para los procesos de compilzación será necesario tener en el mismo directorio no sólo los archivos .jflex y .cup, también los compiladores java-cup-11b-runtime.jar y java-cup-11b.jar cuales permitirán generar las instancias de ejecución.
