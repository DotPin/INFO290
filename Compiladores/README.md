# Disciplina a desplegar

**Compiladores**

***

## Descripción

Disciplina encargada de entregar propuestas traducción a nivel de software, etapas de interpretación para lenguajes de programación a nivel funcional, cuales desempeñadan desde niveles de procesamiento en máquina, tales como: el análisis léxico, análisis sintáctico, análisis semántico, generación de código intermedio, optimización de código y generación de código objeto, entregando una mayor interpretación, funcionalidad, optimización, legibilidad en lo básico.

## Propuesta

Mediante un caso de estudio, se ocuparán los conocimientos adquiridos en la disciplina para crear programas que den cuenta el funcionamiento de las etapas de compilación de un programa de alto nivel, llevándo a uno de bajo nivel de procesamiento.
