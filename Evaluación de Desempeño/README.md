# Disciplina a desplegar

**Evaluación de desempeño**

***

## Descripción

Disciplina cual permite fundamentar los usos de las teorías matemáticas usando de métricas, técnicas, modelos estadísticos para redes y colas, que mediante herramientas ya sea de programación como sistemas informáticos, resuelvan enigmas de rendimientos que surjan a partir de los procesos de monitoreo como de benchmarking, logrando tomar decisiones en las áreas de TI. 


## Propuestas

- Caso de Uso 1: Desafío cual permite presentar una herramienta de benchmarking, cual uso permite probar rendimientos para usos computacionales.

- Caso de Uso 2: Desafío de escojer un sistema computacional cual desempeñe rol de administación, y realizar los análisis de benchmarking/monitoreo según corresponda.
