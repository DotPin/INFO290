# Presentación de Caso 1 de software

**Monitoreo de Sistema Informático**

***

## Necesidad

En este desafío, se realizará un análisis a partir de un sistema informático proveniente de un nodo de red (Hijo), a partir de una infraestructura(Padre), administrada y supervisada por especialistas del área, quienes compartieron datos del servicio de monitoreo.
