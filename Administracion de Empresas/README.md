# Disciplina a desplegar

Administración de Empresas

***

## Descripción

Disciplina encargada de enseñar etapas tanto de planificación como de gestión, para elaboración de propuestas funcionales al interior de una empresa.

## Propuesta

Se hace la presentación de 2 casos de estudio, cuales permiten elaborar propuestas a nivel de diseño a oferentes respectivos.

## Casos de Estudio:

* [Caso 1](https://github.com/DotPin/INFO290/tree/master/Administracion%20de%20Empresas/Caso%201): Presentación a nivel de diseño para una propuesta a nivel de competitividad en el mercado

* [Caso 2](https://github.com/DotPin/INFO290/tree/master/Administracion%20de%20Empresas/Caso%202): Elaboración de un despliegue a nivel funcional de una empresa según sus recursos y funcionalidades.
