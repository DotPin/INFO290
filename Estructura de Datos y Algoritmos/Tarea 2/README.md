# Estudio en características radioeléctricas, de los medios de transmisión inalámbricos.

**Cliente: Dpto de deportes, sección Futbol**

***

## Necesidad

Desafío en creación de prototipo de software en registro de partidos para la Copa Mundial de Futbol Brasil 2014 que se han jugado en la primera fase de grupos, además de conocer el grupo al que pertenece cada equipo y posiciones en tiempo real de
cada uno en la tabla general y por grupo. Para llevar a cabo esta tarea se utilizara lo aprendido sobre estructuras
dinámicas de datos en el lenguaje orientado a objetos C++.
