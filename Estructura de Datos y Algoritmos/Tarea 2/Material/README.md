## Descripción de Datos

Dentro de cada directorio, se almacenan tanto de librerías Python como los archivos fuente que permiten la ejecución del software Compresor, cuales se detallarán a continuación.

- [archivos](https://github.com/DotPin/INFO290/tree/master/Estructura%20de%20Datos%20y%20Algoritmos/Tarea%202/Material/archivos): Directorio cual almacena los archivos exportados del software ocupado

- [Equipos.csv](https://github.com/DotPin/INFO290/blob/master/Estructura%20de%20Datos%20y%20Algoritmos/Tarea%202/Material/Equipos.csv) Archivo cual almacena los equipos de futbol para el software

- [Partidos.csv](https://github.com/DotPin/INFO290/blob/master/Estructura%20de%20Datos%20y%20Algoritmos/Tarea%202/Material/Partidos.csv) Archivo cual almacena los datos de los partidos efectuados con los equipos en competición.

- [equipo.cpp](https://github.com/DotPin/INFO290/blob/master/Estructura%20de%20Datos%20y%20Algoritmos/Tarea%202/Material/equipo.cpp) Código cual describe el la funcionalidad de la librería equipo.h.

- [equipo.h](https://github.com/DotPin/INFO290/blob/master/Estructura%20de%20Datos%20y%20Algoritmos/Tarea%202/Material/equipo.h): Código cual define la funcionalidad delos métodos usados del archivo equipo.cpp

- [main.cpp](https://github.com/DotPin/INFO290/blob/master/Estructura%20de%20Datos%20y%20Algoritmos/Tarea%202/Material/main.cpp): Código cual representa la funcionalidad principal el software desarrollado.

- [partidos.cpp](https://github.com/DotPin/INFO290/blob/master/Estructura%20de%20Datos%20y%20Algoritmos/Tarea%202/Material/partidos.cpp): Código cual define la funcionalidad de los métodos usados del archivo partidos.h

- [partidos.h](https://github.com/DotPin/INFO290/blob/master/Estructura%20de%20Datos%20y%20Algoritmos/Tarea%202/Material/partidos.h): Código cual define la librería paralos métodos usados del archivo partidos.cpp

## Requerimientos de sistema

- SO Linux, Kubuntu 24.04
- Python


### Ejecución en Linux

- Compilar Software

```bash
$make
```

- Ejecutar Software

```bash
$./exec
```


### Ejecución en Windows

Usar programa [DevC++](https://www.dev-cpp.com/), abrir archivo proyecto.dev, compilar, y ejecutar archivo exportado.
